Install in one line.

Requires git, Node (LTS) and pnpm. Updates pull automatically on launch.

macOS & Linux

curl -fsSL https://raw.githubusercontent.com/SaxxSaxx/Nyxcord/main/install.sh | bash

Paste into a terminal and restart Discord.

Windows

irm https://raw.githubusercontent.com/SaxxSaxx/Nyxcord/main/install.ps1 | iex

Run in PowerShell, then restart Discord. Or grab the latest release.

Nyxcord is a fork of Equicord (a Vencord fork). Client mods are against Discord's Terms of Service; they're rarely enforced for cosmetic use, but use at your own risk. Free and open source under GPL-3.0.
